# Code realiseren

Hier is de link van de repository van de Porky-SMP waar de code te vinden is:
- [Porky-SMP](https://github.com/Bit-Academy-Students/PorkySMP-vsb-inc)

In deze repository is de code te vinden die is geschreven voor de Porky-SMP website. 
Dit project is gemaakt in het Engels omdat de projectowner dit graag wilde aangezien de Porky-SMP een internationale Minecraft server is.

## Ontwikkeltools

Dit project is de frontend voor de Porky-SMP website. Hiervoor zijn de volgende ontwikkeltools gebruikt:

- React
- Tailwind CSS


## Project structuur

Het project bestaat uit de volgende pagina's:
- **Home pagina** - Landingspagina met overzicht en introductie van de server
- **Rules pagina** - Uitgebreide regelgeving en richtlijnen voor spelers
- **Wiki pagina** - Overzicht van server informatie en handleidingen
- **News pagina** - Actuele updates en nieuwsberichten over server events
- **Teams pagina** - Overzicht van verschillende teams en hun leden
- **Court pagina** - Juridisch systeem binnen de server met rechtszaken en uitspraken

# Wat heb ik gedaan?

## Design & styling

Tijdens dit project heb ik voornamelijk gewerkt aan het design en de styling aangezien mijn teamgenoot hier minder ervaring mee had. Hierbij was het belangrijk dat het design overzichtelijk, gebruiksvriendelijk en netjes was. Ook moest het design aansluiten bij het kleurenpalet dat we hebben gekregen van de projectowner: roze-, wit- en blauwtinten.

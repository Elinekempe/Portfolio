# Verbetervoorstellen Porky-SMP

## Mobile responsiveness

**Huidige situatie:** 
De website is niet responsive en werkt niet goed op mobiele apparaten. Als je de site opent op een telefoon, is de website moeilijk te navigeren en sommige functies werken niet goed. Dit zorgt ervoor dat de website niet te gebruiken is op mobiele apparaten. Dit zou in de toekomst verbeterd moeten worden.

**Oplossing:** 
In een latere fase van het project kan de website worden aangepast om responsive te zijn. Dit kan gedaan worden door gebruik te maken van CSS en flexbox of grid layout om het design aan te passen aan verschillende schermformaten zodat de website ook op mobiele apparaten goed werkt.

## Zoekfunctie

**Huidige situatie:** 
De News-pagina heeft momenteel geen zoekfunctie. Dit maakt het moeilijk om een specifiek nieuwsartikel te vinden aangezien deze pagina veel artikelen bevat. Het toevoegen van een zoekfunctie zou de snelheid waarmee mensen op de website kunnen zoeken verbeteren en het makkelijker maken om specifieke artikelen te vinden.

**Oplossing:** 
In een latere fase van het project kan een zoekfunctie worden toegevoegd aan de News-pagina. Dit kan worden gedaan door een zoekbalk toe te voegen aan de pagina waarmee gebruikers kunnen zoeken naar een specifiek artikel of onderwerp. ## Rustiger design

**Huidige situatie:**
De website heeft momenteel een druk design met veel kleurverschillen die de aandacht trekken en afleiden van de inhoud. Dit kan het moeilijker maken om belangrijke informatie te vinden. Ook is de spelling en grammatica op sommige pagina's niet correct. Dit kan afleiden van de inhoud. Dit zal in de toekomst verbeterd moeten worden.

**Oplossing:** 
In een latere fase van het project kan het design worden aangepast om het rustiger en overzichtelijker te maken. Dit kan worden gedaan door een beperkter kleurenpallet te gebruiken. Ook kan de spelling en de grammatica op de pagina's worden nagekeken en waar nodig worden verbeterd. Hierdoor wordt de website professioneler en gebruiksvriendelijker.

# Eigen reflectie op het product Porky-SMP

## Toegankelijkheid

Op dit moment is de website niet toegankelijk voor mensen met een visuele, auditieve of motorische beperking. Dit zou moeten worden verbeterd in de toekomst zodat iedereen de website goed kan gebruiken. Hierbij kan worden gedacht aan oplossingen zoals voice-over functionaliteit en een toetsenbord-navigatie. Hierdoor wordt de website toegankelijker en gebruiksvriendelijker voor iedereen en vergroten we het bereik van de website. 

Verbetervoorstellen: 
- Voeg voice-over functionaliteit toe voor blinden en slechtzienden.
- Implementeer toetsenbord-navigatie voor mensen met motorische beperkingen.
- Bij gesproken teksten zorgen dat er een ondertitelingsfunctie is voor mensen met een auditieve beperking.
 
  
## Nette code en documentatie

Momenteel is de code niet altijd netjes en overzichtelijk geschreven. Dit zou moeten worden veranderd in de toekomst door het gebruik van een vaste vorm van naamgeving en het goed bijhouden van documentatie. Hierdoor wordt het later voor developers makkelijker om de code te begrijpen en aan te passen. Het zorgt er ook voor dat de code beter te onderhouden is.

Verbetervoorstellen:
- Gebruik een vaste naamgeving voor variabelen, functies en componenten.
- Houd altijd documentatie van de code bij. 
- Gebruik verschillende bestanden en mappen om de code overzichtelijk te houden. 

## Security en dataveiligheid

Op dit moment is er nog weinig aandacht besteed aan de security en dataveiligheid van de website. Dit aspect moet in de toekomst nog worden verbeterd. Dit zou kunnen door de functie toe te voegen dat bepaalde knoppen of pagina's niet beschikbaar of grayed-out zijn voor mensen die niet de juiste permissies hebben. Daarnaast moet er ook aandacht worden besteed aan de bescherming van gebruikersdata door het implementeren van een privacybeleid.

Verbetervoorstellen:
- Implementeer permissies voor verschillende gebruikersrollen.
- Voeg beveiligingsmaatregelen toe zoals HTTPS.
- Zorg voor veilige communicatie tussen de website en de server.

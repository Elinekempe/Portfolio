## Opdracht 2: Minesweeper Game

Dit is een opdracht die ik zelf heb bedacht en uitgewerkt. Hierin heb ik een minesweeper game gemaakt met verschillende technische onderdelen. 

De bedoeling is dat de speler een veld met verborgen mijnen moet zien te overleven door vakjes te openen zonder op een mijn te klikken. Het doel is om alle niet-mijn vakjes te openen. Ook kan je verschillende niveaus kiezen en wordt je tijd bijgehouden.
